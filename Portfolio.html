Featured Project

Speed Slalom Exploratory Research Project

The goal of this project is to investigate the correlation between angular velocity, linear acceleration, and final sprint speed in elite national-level speed slalom athletes. Project Overview Speed slalom is a high-intensity skating discipline where athletes race through a line of cones with maximum speed and precision. Despite its growth as a competitive sport, there is limited scientific research on its biomechanics.

Sentiment Analysis with EmoAtlas

Worked on an emotion analysis project using the EmoAtlas library to process DOCX documents and extract emotional patterns from text. The system filtered and analysed sentences based on keywords related to weight, exercise, and lifestyle, enabling structured identification of sentiment trends within large bodies of written content. This project combined natural language processing with data filtering techniques to better understand emotional framing in health-related narratives.
